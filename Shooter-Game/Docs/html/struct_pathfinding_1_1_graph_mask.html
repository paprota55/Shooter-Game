Detailed Description

Holds a bitmask of graphs.

This bitmask can hold up to 32 graphs.

The bitmask can be converted to and from integers implicitly.

GraphMask mask1 = GraphMask.FromGraphName("My Grid Graph"); GraphMask mask2 = GraphMask.FromGraphName("My Other Grid Graph");

NNConstraint nn = NNConstraint.Default;

nn.graphMask = mask1 | mask2;

// Find the node closest to somePoint which is either in 'My Grid Graph' OR in 'My Other Grid Graph' var info = AstarPath.active.GetNearest(somePoint, nn);

See: bitmasks (view in online documentation for working links)
